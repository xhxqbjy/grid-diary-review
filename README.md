## 格志日记是什么？

格志日记（Grid Diary）是一款来自国内团队的日记 App，2013 年就开始了，陪着好几百万用户写到了现在。支持 iOS、Android、macOS 三端，数据可以跨设备同步。

它最核心的设计就是"九宫格"——把一页日记切成几个格子，每个格子配一个引导问题。你不用想从哪里下笔，按格子回答就好。

这个设计乍听简单，用起来真的挺有意思。

---

## 为什么用格子，而不是空白页？

空白页的问题在于：它要求你有"写作意愿"才能开始。而格子的好处是，它把"写日记"这件事拆成了几个小问题，每个问题都很具体，很容易回答。

格志内置的问题库基于积极心理学设计，不是随便问问。"今天最好的事情是什么"这种问题，会让你主动去回想这一天有没有什么值得记录的瞬间。时间长了，这个习惯本身就在改变你看待生活的方式。

当然，你也可以完全自定义问题。内置的问答只是个起点，你可以删掉不喜欢的，加入自己想追踪的内容——运动打卡、花销记录、睡眠质量，都行。
